# Changelog

## 0.1.0

H5 Forge 首个公开版本，基于 Flutter Forge 的工作流形态改造为 H5/Web 项目协作 skill。

- 新增 H5/Web 项目主控 `SKILL.md`
- 支持“硬排除 + 运维直通 + 轻量执行 + 完整流程”的任务路由
- 支持已有 H5/Web 项目接入和新 H5/Web 应用 `mvp / business / team` profile
- 支持规则卡、会话状态、相似实现检索、输入不完整处理和完整流程升级原因
- 新增前端项目质量、路由、网络层、测试、i18n/a11y、代码审查和迁移参考文档
- 新增 MIT License、贡献指南、GitHub issue 模板和开源发布检查清单
- 新增 `scripts/discover_h5_skills.sh`，用于探测本地可协作的前端 skills
